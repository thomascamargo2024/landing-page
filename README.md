# 🚀 Landing Page - Projeto Web

Este é um projeto de **Landing Page moderna** desenvolvido com **HTML, CSS e JavaScript**, com foco em design responsivo, interatividade e boas práticas de front-end.

## ✨ Sobre o Projeto

Esta landing page foi criada com o objetivo de praticar e demonstrar habilidades em desenvolvimento web front-end, incluindo:

- Estruturação semântica com **HTML5**
- Estilização moderna com **CSS3**
- Interatividade com **JavaScript puro**
- Layout responsivo para diferentes dispositivos

---

## 🎯 Funcionalidades

- Layout responsivo (mobile e desktop)
- Design moderno e limpo
- Animações e interações com JS
- Seções organizadas (hero, sobre, serviços, contato, etc.)

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
